PHP Car Shop
Overview

The PHP Car Shop is a web-based application that allows users to browse and interact with a car dealership’s inventory.
The project demonstrates full-stack development using PHP, HTML, CSS and MySQL, along with features like
dynamic product galleries, user login, and admin dashboard overview.

It is designed as a student project but can be extended into a real-world dealership platform.

## Technologies Used

Backend: PHP, MySQL

Frontend: HTML, CSS

Server Environment: XAMPP
